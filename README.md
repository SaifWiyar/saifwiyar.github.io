## Environment Variables

Do not upload a real `.env` or `.env.local` file to GitHub.

The public repository contains:

```text
.env.example
```

Example:

```env
VITE_FORMSPREE_ID=
```

For local development, create:

```text
.env.local
```

Add your Formspree form ID:

```env
VITE_FORMSPREE_ID=your_formspree_form_id
```

Example:

```env
VITE_FORMSPREE_ID=abcdwxyz
```

The `.env.local` file is ignored by Git and must remain private.

## Contact Form Setup

### Recommended Option: Formspree

Formspree is recommended because the website is hosted on GitHub Pages, which does not provide a backend server.

1. Create a Formspree account.
2. Create a new form.
3. Copy the form ID.
4. Add it locally to `.env.local`.
5. Add it to GitHub Actions as a repository variable.

GitHub location:

```text
Repository
→ Settings
→ Secrets and variables
→ Actions
→ Variables
→ New repository variable
```

Variable name:

```text
VITE_FORMSPREE_ID
```

Variable value:

```text
your_formspree_form_id
```

Do not place private keys directly inside React components.
